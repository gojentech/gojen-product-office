## What memory is

Memory files are durable, curated summaries that help AI assistants and humans reload shared context quickly. They are **not** the single source of truth for product requirements. When memory and product folders disagree, product folders win until memory is updated.

## Update rules

- Keep entries factual and link to SSOT paths.
- Do not paste entire standards or product PRDs into memory — summarize and link.
