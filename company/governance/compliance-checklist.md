## Document Compliance (every new/updated GAIOS doc)

- [ ] Document ID present and unique
- [ ] Document Name, Version `1.0.0+`, Status set correctly
- [ ] Owner, Reviewer, Approver named
- [ ] Created Date and Last Updated set
- [ ] Purpose and Scope stated
- [ ] Navigation: Parent, Child, Related, Previous, Next, Back to START-HERE
- [ ] No placeholder / lorem text
- [ ] No real customer PII
- [ ] Cross-links resolve

## Decision Compliance

- [ ] Decision ID, Owner, Problem Statement present
- [ ] Options Considered with trade-offs
- [ ] Decision, Reason, Impact clear
- [ ] Status, Approval, Review Date recorded
- [ ] REGISTER.md row added/updated

## Risk Compliance

- [ ] Risk ID, Category, Probability, Impact present
- [ ] Mitigation and Owner present
- [ ] Current Status accurate
- [ ] REGISTER.md row added/updated

## AI Session Compliance

- [ ] Session used [AI Session Template](../templates/ai-session-template.md) or equivalent fields
- [ ] Outputs verified before external use
- [ ] Promotion list created for SSOT updates
- [ ] Agent brief used for multi-step agents
- [ ] AI did not self-approve decisions

## Portfolio / Product Claims Compliance

- [ ] Subscription OS vs Pawn Management scope not conflated
- [ ] Pricing/compliance claims match Approved decisions or marked hypothesis
- [ ] Jurisdiction statements for pawn ops reviewed before public use
- [ ] Payment/PCI statements do not overclaim
